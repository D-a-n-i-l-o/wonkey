Welcome to wonkey!

wonkey is an open-source, user-friendly, cross-platform programming language designed primarily for game programming.

All documentation is currently hosted online at github.com/wonkey-coders/wonkey.

Several sample wonkey apps can be found in the 'examples' directory of the wonkey release.

The wonkey release includes a VERY simple IDE coded in pure wonkey called 'wide'. 'wide' is not a fully fledged IDE (yet) but is intended to provide a 'works anywhere' IDE solution for using wonkey on any platform you can get it building on.

So a HUGE shout out to everyone who has contributed to wonkey (formerly known as monkey2) to date! wonkey would not have been possible without your generosity - thank you very much!
